# emSmartUpScale
Gizmo to quick to upscale renders without using AI. This technique is from a LinkedIn post, and I wanted to turn it into a Gizmo.
![emSmartUpScale_Gizmo](https://github.com/user-attachments/assets/8c25dfe7-68ad-4493-ac6f-d925e4aa5f57)
## How to use it
* I would suggest you put in your end of your comp. It's using the Lanczos6 filter, and it will destroy your TechPass if you use it after loading your Read file.
