# emSmartUpScale
Gizmo to quick to upscale renders without using AI. I learnt this technique in a LinkedIn post, and I wanted to turn it into a Gizmo.

## How to use it
* I would suggest you put in your end of your comp. It's using the Lanczos6 filter, and it will destroy your TechPass if you use it after loading your Read file.
